# Accessibility checklist (initial)

- Ensure all Streamlit controls have clear text labels.
- Verify focus order using keyboard navigation; add `st.help` and hints where needed.
- Check color contrast on overlays and status pips; provide alternative themes.
- Provide keyboard shortcuts for timeline controls (already added) and document them.
- Add ARIA attributes where applicable in custom HTML components.
- Document remaining gaps here with screenshots.
